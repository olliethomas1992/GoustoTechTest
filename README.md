# Ollie Thomas's Gousto Tech Test

## Install and Setup

First run:

```
npm install
```

Once all the packages are installed run

```
npm start
```

to run the application.

Or 

```
npm run build
```

to create a production build.

## Testing

Run
```
npm test
```

to run unit tests

or

```
npm run coverage
```

to view the coverage report.
